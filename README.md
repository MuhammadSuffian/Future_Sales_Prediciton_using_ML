# Future Sales Prediction using Linear Regression

This project predicts future sales using the Linear Regression algorithm in Python. The model is trained on historical advertising and sales data to understand the relationship between marketing expenses and product sales.

## Features
- Data preprocessing and analysis
- Linear Regression model training
- Sales prediction on new data
- Model evaluation using performance metrics
- Data visualization for better insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
The dataset contains advertising spending on:
- TV
- Radio
- Newspaper

Target Variable:
- Sales

## Results
The model learns the relationship between advertising budgets and sales, allowing future sales predictions based on marketing investments.
